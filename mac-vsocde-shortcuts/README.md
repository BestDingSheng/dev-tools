# mac vscode 快捷键

整理些比较实用但是我自己总是记不住的一些快捷键

- `command + ~` 切换 vscode 窗口
- `command + k + 0` 代码折叠
- `command + k + j` 展开代码
- `options + shift + 鼠标点击` 选择多行
- `control + ~` 打开终端
- `options + z` 自动换取消自动换行
- `command + shift + k` 删除当前行
